Console API documentation
Console methods overview
Methods Description
console.info() Informative logging output with info icon in front.
console.log() Output a Logging message without icon.
console.dir() Displays an interactive listing of the properties of a specified JavaScript object. This listing lets you use disclosure triangles to examine the contents of child objects.
console.debug() Alias for console.log.
console.warn() Outputs a warning message with warn icon in front.
console.error() Outputs an error message with error icon in front.
console.assert() Log a message and stack trace to console if first argument is false
console.count() Log the number of times this line has been called with the given label.
console.group() Creates a new inline group, indenting all following output by another level. To move back out a level, call groupEnd()
console.groupCollapsed() Creates a new inline group, indenting all following output by another level; unlike group(), this starts with the inline group collapsed, requiring the use of a disclosure button to expand it. To move back out a level, call groupEnd().
console.groupEnd() Exits the current inline group
console.clear() Clear the console
console.time() Starts a timer with a name specified as an input parameter. Up to 10,000 simultaneous timers can run on a given page
console.timeEnd() Stops the specified timer and logs the elapsed time in seconds since its start.
console.table() Clear the console
console.profile() Starts recording a performance profile
console.profileEnd() The profileEnd method stops recording a profile previously started
console.trace() Prints a stack trace from the point where the method was called
console.timeStamp() Adds an event to the Timeline during a recording session
Using string substitutions
Substitution
string		 Description
%o or %O Outputs a JavaScript object. Clicking the object name opens more information about it in the inspector.
%d or %i Outputs a digit/integer. Number formatting is supported, for example console.log("Foo %.2d", 1.1) will output the number as two significant figures with a leading 0: Foo 01
%s Outputs a string.
%f Outputs a floating-point value. Formatting is supported, for example console.log("Foo %.2f", 1.1) will output the number to 2 decimal places: Foo 1.10
%c Use the %c directive to apply a CSS style to console output.

console.log("This is %cMy stylish message", "color: yellow; font-style: italic; background-color: blue;padding: 2px");

This is My stylish message

console.info

Informative logging output with info icon in front. You may use string substitution with this method.

Syntax

console.info(msg [, msg1, ..., msgN]);

Example

console.info('Mr. Dump is not', 2, 'smart');

Output

Mr. Dump is not 2 smart

console.log

Output a Logging message without icon. You may use string substitution with this method.

Syntax

console.log(msg [, msg1, ..., msgN]);

Example

console.log('Mr. Dump is not', 2, {property: 'smart'});

Output

Mr. Dump is not 2 Object {property: "smart "}

console.dir

Displays an interactive list of the properties of the specified JavaScript object. The output is presented as a hierarchical listing with disclosure triangles that let you see the contents of child objects.
Best use for console.dir is for object contain DOM elements.

  • console.log prints the element in an HTML-like tree
  • console.dir prints the element in a JSON-like tree

Syntax

console.dir(object);

Example

console.dir({property: 'smart'});

Output

Object

console.warn

Outputs a warning message with warn icon in front. Useful when informating the user on usage deprecated methods. You may use string substitution with this method.

Syntax

console.warn(msg [, msg1, ..., msgN]);

Example

console.warn('Mr. Dump is not %d smart.', 2);

Output

Mr. Dump is not 2 smart

console.error

Outputs a warning message with warn icon in front. Useful when informating the user on usage deprecated methods. You may use string substitution with this method.

Syntax

console.error(msg [, msg1, ..., msgN]);

Example

console.error('Mr. Dump is not %d smart.', 2);

Output

Mr. Dump is not 2 smart

console.assert

Writes an error message to the console if the assertion is false. If the assertion is true, nothing happens. You may use string substitution with this method.

Note: Specifically, in browsers, calling console.assert() with a falsy assertion will cause the message to be printed to the console without interrupting execution of subsequent code. In Node.js, however, a falsy assertion will cause an AssertionError to be thrown.

Syntax

console.assert(assertion, msg1 [, msg2, ..., msgN]);

Example

function greaterThan(a,b) {
  console.assert(a > b, "a (%d) is not greater than b (%d)", a, b});
}
greaterThan(5,6);

Output

a (5) is not greater than b (6)

console.table

This function takes one mandatory argument data, which must be an array or an object, and one additional optional parameter columns.

Note that (in Firefox) console.table is limited to displaying 1000 rows (first row is the labeled index).

Note that in Google Chrome console.table require data with 2 dimensions. The first represent always the index.

Syntax

console.table(data [, [column1, ..., columnN] ]);

Example 1

// array of strings
console.table([ ["apples", "oranges", "bananas"] ]);

Output

(index) 0 1 2
0 "apples" "oranges" "bananas"

Example 2

function Person(firstName, lastName, age) {
    this.firstName = firstName;
    this.lastName = lastName;
    this.age = age;
}

var family = {};
family.mother = new Person("Susan", "Doyle", 32);
family.father = new Person("John", "Doyle", 33);
family.daughter = new Person("Lily", "Doyle", 5);
family.son = new Person("Mike", "Doyle", 8);

console.table(family, ["firstName", "age"]);

Output

(index) firstName age
mother "Lea" 32
father "Jan" 33
daughter "Julia" 6
son "Nicolas" 15

console.clear

Clears the console.
If in Chrome the Preserve log checkbox is enabled, console.clear() is disabled.

Syntax

console.clear();

console.count

Writes the number of times that count() has been invoked at the same line and with the same label. This function takes an optional argument label

Syntax

console.count();

Example

function login(name) {
  console.count(name + ' logged in');
}

login("Bob");
login("Mike");
login("Bob");
login("Bob");

Output

Bob logged in: 1
Mike logged in: 1
Bob logged in: 2
Bob logged in: 3

console.time

Starts a new timer. Call console.timeEnd() to stop the timer and print the elapsed time to the Console.

Pass an optional label to change the output text that precedes the elapsed time. Call console.timeEnd() with the same label to stop the timer.
Use labels to run multiple timers at the same time.

Syntax

console.time([label]);
...
console.timeEnd([label]);

Example

console.time();
var arr = new Array(10000);
for (var i = 0; i < arr.length; i++) {
  arr[i] = new Object();
}
console.timeEnd();

Output

default: 3.35ms

Example multiple usage

console.time('total');
console.time('init arr');
var arr = new Array(10000);
console.timeEnd('init arr');
for (var i = 0; i < arr.length; i++) {
  arr[i] = new Object();
}
console.timeEnd('total');

Output

init arr: 0.0720ms
total: 4.34ms

console.group

Starts a new group. Call console.groupEnd() to stop the timer and print the elapsed time to the Console.
You can also nest groups.

To display created group collapsed use console.groupCollapsed

Syntax

console.group(msg [, msg1, ..., msgN]);
...
console.groupEnd(msg [, msg1, ..., msgN]);

Example

function name(obj) {
    console.group('name');
    console.log('first: ', obj.first);
    console.log('middle: ', obj.middle);
    console.log('last: ', obj.last);
    console.groupEnd();
}

name({"first":"Wile","middle":"E","last":"Coyote"});

Output

name
first: Wile
middle: E
last: Coyote

Example nested usage

function name(obj) {
    console.group('name');
    console.log('first: ', obj.first);
    console.log('middle: ', obj.middle);
    console.log('last: ', obj.last);
    console.groupEnd();
}

function doStuff() {
    console.group('doStuff()');
    name({"first":"Wile","middle":"E","last":"coyote"});
    console.groupEnd();
}

doStuff();

Output

doStuff()
name
first: Wile
middle: E
last: Coyote

console.groupCollapsed

Creates a new inline group in the Web Console. Unlike console.group(), however, the new group is created collapsed. The user will need to use the disclosure button next to it to expand it, revealing the entries created in the group.

Syntax

console.groupCollapsed(msg [, msg1, ..., msgN]);
...
console.groupEnd(msg [, msg1, ..., msgN]);

console.timeStamp

Adds a single marker to the browser's Timeline or Waterfall tool. This lets you correlate a point in your code with the other events recorded in the timeline, such as layout and paint events.

You can optionally supply an argument to label the timestamp, and this label will then be shown alongside the marker.

Syntax

console.timeStamp([label]);

console.profile

Starts a JavaScript CPU profile with an optional label. To complete the profile, call console.profileEnd(). Each profile is added to the Profiles panel.

Syntax

console.profile([label]);

Example

function processPixels() {
    console.profile("processPixels()");
    // later, after processing pixels
    console.profileEnd();
}
